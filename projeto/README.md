# Habit Tracker - Projeto WebMAC

Este é um projeto de rastreamento de hábitos desenvolvido para a primeira fase da disciplina de Introdução ao Desenvolvimento de Sistemas de Software (MAC0350) . O aplicativo permite criar, gerenciar e buscar hábitos diários.

## Funcionalidades (Requisitos)

    * Criar novos hábitos com descrição e tags. (hx-post)
    * Buscar e Listagem de hábitos. (hx-get)
    * Atualizar o progresso do hábito no banco de dados. (hx-put)
    * Excluir hábitos.(hx-delete)
    * Modelos Habit e Tag com relação many-to-many
